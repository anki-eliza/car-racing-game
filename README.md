# car-racing-game
This is a simple car racing game developed using Python's PyGame library.The goal is to dodge incoming cars, survive as long as possible, and achieve the highest score!

##How to Play
###1.Start Screen
-Press ENTER to start the game.
-Press Q to quit.

###2.Controls
-LEFT Arrow Key : Move your car to the left
-RIGHT Arrow Key : Move your car to the right

###3.Scoring
-Score increases each time you dodge an obstacle car.
-Game speed increases every time your score crosses a multiple of 5.

###4.Game Over
-If you collide with an obstacle, the game ends.
-Press Y → Play again
-Press N → Exit


Instructions to play the game:
1.On the start page, press ENTER Key to start the game or press Q to quit the game.
2.Control the player car using LEFT or RIGHT Arrow keys to move the player car left or right respectively.
3.Score is updated for every obstacle car that you manage to dodge.
4.Speed of the game increases as score crosses a multiple of 5, making the challenge harder.
5.When the player car collides with an obsctacle car, the game ends and prompts the user to play again by pressing Y  or N to exit.
